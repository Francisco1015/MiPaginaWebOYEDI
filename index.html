Estructura y Componentes de Internet

Internet es una red de redes. Está compuesta por componentes físicos como servidores, routers, cables y enlaces satelitales, y por protocolos lógicos como TCP/IP que permiten el flujo de datos entre dispositivos interconectados a nivel global.
